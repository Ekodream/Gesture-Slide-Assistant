# AGENTS.md — 手势控制演示助手

本项目的中文开发规范以 `AGENTS_zh.md` 为准。实施时必须遵守以下要点：

- 按阶段推进，不要一次性重写整个项目；
- 保留 `--dry-run` 行为；
- PyAutoGUI 真实动作只允许在非 dry-run 模式下执行；
- MVP 阶段不得自动鼠标点击；
- 不增加不必要依赖；
- 不实现模型训练、复杂 GUI、数据库、云服务或大型框架；
- 函数保持小型，适当添加类型标注；
- 优雅处理无手势和摄像头失败；
- 修改后运行 `pytest`，硬件功能使用真实摄像头人工验证；
- 禁止使用批量删除命令。需要删除文件时，只能一次删除一个具体路径。
