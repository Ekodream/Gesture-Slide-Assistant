# 人工测试计划

摄像头、MediaPipe 检测、真实翻页和指针移动必须使用真实硬件测试，不使用 fake 摄像头。

| 测试 | 操作 | 预期结果 | 实际结果 |
|---|---|---|---|
| 下一页 | 做 `V_SIGN` | 文档前进一页 | 待填写 |
| 上一页 | 做 `THREE_FINGERS` | 文档后退一页 | 待填写 |
| 开始/暂停 | 做 `FIST` | 发送配置按键 | 待填写 |
| 指针 | 做 `INDEX_ONLY` | 指针移动 | 待填写 |
| 无手 | 手离开画面 | 不触发命令 | 待填写 |

## 环境变化测试

至少选择两类填写：

| 环境变化 | 操作 | 预期结果 | 实际结果 |
|---|---|---|---|
| 不同光照 | 在较亮/较暗环境中测试四种手势 | 手势仍可识别，误触发可接受范围内 | 待填写 |
| 不同手部距离 | 手靠近/远离摄像头 | 检测稳定，不崩溃 | 待填写 |
| 不同背景 | 更换纯色/复杂背景 | 手势仍可识别 | 待填写 |
| 更快/更慢的手势变化 | 快速切换和缓慢保持手势 | 防抖能阻止短暂误触发 | 待填写 |
